# RUNAWORLD

---

## Descripcion

RUNAWORLD es un juego basado en navegador donde los jugadores lanzan runas interactuando con un boton central, activando animaciones y efectos especiales cuando aparecen runas raras. El juego combina animaciones en Canvas y SVG para crear una respuesta visual dinamica, enfocandose en mecanicas simples con resultados atractivos.

## Caracteristicas

* Sistema de lanzamiento de runas con diferentes niveles de rareza.
* Animaciones dinamicas mediante Canvas y SVG.
* Gestion de recursos y multiplicadores de suerte.
* Interfaz de usuario minimalista y funcional.
* Interfaz de administrador minimalista y funcional.
